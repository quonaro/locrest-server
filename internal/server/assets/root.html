Locrest Tunnel

Expose your local servers to the internet in seconds.

1. Pick a local port
Replace 8080 with the port your app is running on locally.
2. Run the one-liner
Copy the command below and paste it into your terminal.
Create a public tunnel
# Replace 8080 with your local port
curl -s https://{{.Domain}}/8080 | bash
Create an authenticated tunnel
# Replace 8080 with your local port and paste your API token
curl -H "Authorization: Bearer <token>" -s https://{{.Domain}}/8080?infinity=true | bash
Client authentication
The locrest-client binary does not accept the API token directly. The one-liner above returns a script that downloads the client and passes it a temporary setup-token. The client then performs an ED25519 challenge-response handshake using that setup-token.
